## Why This Project?

This is a simple Q&A (Question & Answer) website. Any user can:
- Ask questions
- Answer other users' questions
- Browse questions by category

---

##  Features

| Feature | Description |
|---------|-------------|
|  **User Auth** | Create new account and login |
|  **Ask Questions** | Ask questions on any topic |
|  **Answer** | Answer other users' questions |
|  **Categories** | Questions organized by categories |
|  **Delete** | Delete your own questions |
